# Installation

## Voraussetzungen
- Python 3.10+
- Minecraft 1.21.8 (Fabric)
- Fabric Loader 0.16.0+

## Mod installieren

1. `mlang-mod-1.0.0.jar` herunterladen
2. In den `mods/` Ordner legen
3. Port/Token in `config/mlang.json` konfigurieren

## Python-Paket installieren

```bash
cd mlang
pip install -e .
```

## Starten

```bash
python -m mlang script.glang
python -m mlang --repl
```
